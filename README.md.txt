# 🌐 Real-Time Speech & Text Translator

A real-time translator built with Streamlit that supports:

- 🗣️ Speech-to-Speech Translation
- 💬 Text-to-Text Translation
- 🌍 Auto Language Detection
- 📥 Downloadable Translations
- 🕘 Translation History with Side-by-Side View

## 🚀 Features
- Google Translate integration via `googletrans`
- Audio playback via `gTTS` and `pygame`
- Streamlit UI with multilingual support
- Clean and minimal design

## 💻 How to Run

```bash
pip install -r requirements.txt
streamlit run main.py
